🎨 Cradle Play School Website

A modern, responsive, and engaging Kids Play School Website built using React + TypeScript, designed to provide a joyful and interactive experience for parents and children.

🚀 Live Demo

🔗 https://cradel-play.netlify.app/

📌 Project Overview

This project is a fully responsive school website focused on:

Showcasing play school activities 🎈
Providing admission information 📚
Creating a friendly and colorful UI for kids 👶
Ensuring smooth navigation for parents 👨‍👩‍👧
🛠️ Tech Stack
⚛️ React.js – Component-based UI
🔷 TypeScript – Type safety & scalability
🎨 CSS / Tailwind / Custom Styling – Modern UI design
🌐 Netlify – Deployment
🧩 Additional Tools:
UI/UX design crafted manually ✍️
Responsive layout techniques 📱
Animation & visual enhancements ✨
✨ Features
🌈 Beautiful and colorful kids-friendly UI
📱 Fully responsive (Mobile + Tablet + Desktop)
⚡ Fast loading and optimized performance
🧭 Smooth navigation and structured layout
🎯 Clean and reusable components
🎨 Custom design (No templates used)

⚙️ Installation & Setup

Follow these steps to run locally:

# Clone the repository
git clone https://github.com/ParasSharma13/cradle-play-school.git

# Navigate to project folder
cd cradle-play-school

# Install dependencies
npm install

# Run the project
npm run dev
